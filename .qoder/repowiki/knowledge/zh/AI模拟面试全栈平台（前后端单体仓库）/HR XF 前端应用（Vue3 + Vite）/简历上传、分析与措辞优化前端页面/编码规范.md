- 所有视图在 `onMounted` 中并行调用 `Promise.all([resumeStore.fetchResume(), resumeStore.fetchConfig()])` 初始化数据与 API Key 配置。
- UI 状态严格受控于 store 的布尔标志位（`loading`/`analyzing`/`optimizing`/`hasApiKey`），模板通过 `v-if` 分支切换按钮禁用态与加载文案。
- 错误处理统一以 `alert` 抛出 `Error.message`，并在 `finally` 块中复位 loading 标志，避免 UI 卡死。
- SSE 流式回调采用三段式 `(onItem, onDone, onStart?)` 形参约定，由 store 内部按 `type=start|item|done` 分发，视图层只关心业务追加逻辑。
