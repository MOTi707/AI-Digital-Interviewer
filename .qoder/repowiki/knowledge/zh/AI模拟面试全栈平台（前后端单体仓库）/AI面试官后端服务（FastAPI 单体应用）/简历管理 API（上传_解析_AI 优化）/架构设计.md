模块按 FastAPI 标准三层划分：
- `routers/resume.py`：HTTP 层，定义 `/api/resume` 路由组，负责文件上传（`UploadFile` + `Form`）、参数校验、异常转 HTTPException、以及通过 `StreamingResponse` 暴露 SSE 端点；依赖 `get_db` / `get_current_user` 注入。
- `services/resume_service.py`：业务层，封装 SQLAlchemy 异步 CRUD（`get_resume_by_user`、`save_resume` UPSERT、`update_parsed_content`）与 Deepseek 调用（`call_deepseek` 同步 + `optimize_wording_stream` 流式），通过 `app.config.get_settings()` 读取 `.env` 中的 `DEEPSEEK_API_KEY/URL/MODEL`。
- `schemas/resume.py`：Pydantic v2 请求/响应模型，`ResumeResponse` 设置 `from_attributes=True` 以直接映射 ORM 对象。

依赖方向单向：router → service → (SQLAlchemy model + httpx)；router 不直接访问 DB 或外部 API。上传目录固定为 `backEnd/uploads/resumes`，使用 PyMuPDF (`fitz`) 在服务端做 PDF 文本提取，避免前端 pdf.js 的不稳定性。AI 功能均受 `has_api_key()` 开关保护，未配置时返回 400。
