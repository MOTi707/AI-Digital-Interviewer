本模块采用三层切分：
- `routers/admin.py`：仅暴露 `/api/admin/*` 的 FastAPI 路由，每个端点通过 `Depends(_require_admin)` 做简易 RBAC（邮箱或用户名包含 `admin`），再依赖 `get_db` 注入 `AsyncSession`，随后委托给 `app.services.admin_service`。
- `services/admin_service.py`：纯异步函数集合，使用 SQLAlchemy 2.0 `select()` + `func.count()` 实现分页查询与 CRUD，不感知 HTTP 协议；返回 ORM 实体或 dict，由路由层用 Pydantic schema 校验后再响应。
- `dependencies.py`：集中定义全局 FastAPI 依赖——`HTTPBearer` + `decode_access_token` + `get_user_by_id` 组合成 `get_current_user`，被 `_require_admin` 复用。
依赖方向单向：router → service → models/database；router 不直接 import models，只通过 service 和 schemas 交互。
