- 路由函数统一采用 `Depends(get_current_user)` / `Depends(_optional_user)` + `Depends(get_db)` 注入当前用户与异步数据库会话，区分受保护与公开接口。
- HTTP 异常通过捕获 service 抛出的 `PermissionError` / `ValueError` 再包装为 `HTTPException(status_code=403|404, detail=str(e))` 向上抛出，保持路由层错误语义化。
- 列表接口返回统一的 `{posts/comments, total, page, size}` 分页结构，由 `PostListResponse` / `CommentListResponse` 约束。
- ORM→Response 转换集中在 service 层的 `_build_*_response` 辅助函数中，避免在 router 里重复拼装字段。
- Pydantic 模型统一启用 `model_config = {"from_attributes": True}` 以便直接从 SQLAlchemy ORM 实例序列化。
